EightyPlus Privacy Policy

EightyPlus is a family of Chrome Extensions that helps users read news briefings, observe broker or exchange market data, save selected cards or contracts, and ask AI-assisted questions about selected read-only context. This Privacy Policy explains what data the extension stores, what data may be sent to third-party services, and how users control that data.

Data Stored Locally

EightyPlus stores the following data locally in the user's browser through Chrome extension storage:

  • Extension settings such as theme, language, and selected AI provider.
  • User-provided API keys for search, AI, broker, or exchange providers.
  • Saved news cards, saved groups, and user-written notes attached to saved cards.
  • Saved broker or exchange watchlists, scanner presets, condition formulas, and recent read-only scan results.
  • Current chat context and chat messages used inside the extension panel.
  • Optional read-only account, asset, portfolio, position, or contract snapshots when the user chooses to load them.

Users may also export chat files or saved card JSON files. Exported files are saved only where the user chooses on their own device.

Data Sent to Third-Party Services

EightyPlus sends data to third-party services only when needed for a user-requested feature. Depending on the user's settings and actions, this may include:

  • Briefing and news requests to the configured EightyPlus news backend.
  • Search queries to Tavily or DuckDuckGo fallback search.
  • News titles, summaries, prompts, and chat messages to the AI provider selected by the user.
  • Selected broker or exchange symbols, market tickers, scanner requests, public market data requests, and optional read-only account snapshot requests to the corresponding broker or exchange API chosen by the user.
  • Subscription or billing requests to the configured billing provider, such as Lemon Squeezy, if paid access is enabled.

Third-party providers process data according to their own privacy policies. Users should review the privacy policy of any AI, search, broker, exchange, or billing provider they choose to use with EightyPlus.

API Keys

User-provided API keys are stored locally in Chrome extension storage. EightyPlus uses those keys only to send requests to the corresponding provider selected by the user. EightyPlus does not sell API keys and does not intentionally share them with unrelated services.

Broker and exchange API keys are optional unless the user chooses a feature that needs private read-only data. Public market data features may work without broker or exchange API keys where the provider allows public access. API keys and secrets are not included in Chat or Deep Research context unless the user explicitly types them into a prompt.

Broker and Exchange Data

Depending on the installed EightyPlus product, the extension may connect to supported providers such as Kraken, Binance, Alpaca, Upbit, or Kalshi. The extension uses these connections for read-only observation, such as public market snapshots, watchlists, scanner results, optional account or position snapshots, and live market updates where available.

EightyPlus does not provide order placement, trading execution, account transfers, deposits, withdrawals, or other account-changing actions. Broker and exchange context may be attached to Chat or Deep Research only as read-only background when the user chooses that workflow.

Browsing History

EightyPlus does not collect full browsing history. When the user opens or saves a news card, the extension may store or use the URL associated with that selected news item so the user can revisit or export it.

Payments and Subscriptions

If subscription features are enabled, payment details are processed by the configured payment provider. EightyPlus stores only the subscription or license status needed to unlock paid functionality. EightyPlus does not store full payment card numbers.

Data Sharing and Sale

EightyPlus does not sell personal data. Data is shared only as needed to provide user-requested features, comply with legal obligations, protect against abuse, or operate the extension.

Retention and Deletion

Data stored in Chrome extension storage remains on the user's device until the user deletes it, clears extension data, or uninstalls the extension. Exported files remain wherever the user saved them and can be deleted by the user at any time.

Security

EightyPlus uses Chrome extension storage and HTTPS requests where supported by the provider. No method of storage or transmission is perfectly secure, so users should protect their device and avoid entering API keys they do not want used by the extension.

Children

EightyPlus is not directed to children under 13 and is intended for users who can evaluate news, AI output, and investment-related information responsibly.

Changes to This Policy

This policy may be updated when EightyPlus features, providers, or data handling practices change. The latest version will be published on this page.
